# Champagne Sacred Assets Guard

These files are **canon** and must NOT be modified, optimised, colour-shifted,
or replaced by any automation (Codex, agents, scripts) without an explicit,
human-reviewed “Sacred Hero Upgrade” brief.

Protected directories:

- public/assets/champagne/motion/**
- public/assets/champagne/particles/**
- public/assets/champagne/textures/**
- public/assets/champagne/waves/**
- public/brand/waves/**
- packages/champagne-hero/**
- packages/champagne-manifests/data/champagne_machine_manifest_full.json
- packages/champagne-manifests/reports/manifest-hero-section-map.md

Allowed operations by automation:

- **Read-only** for analysis, theming, layout wiring.
- **Import/use** in components and layouts.

Forbidden operations by automation:

- Deleting, moving or renaming these files.
- Re-encoding or compressing their contents.
- Replacing gradient, wave, grain or particle assets with new files.
- Changing Sacred Hero gradient laws or motion parameters.

Any PR touching these paths MUST be created manually and reviewed by Nicko.
