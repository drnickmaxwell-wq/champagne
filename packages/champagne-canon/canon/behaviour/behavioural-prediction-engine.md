# Champagne Behavioural Prediction Engine

Predicts impending overwhelm and simplifies the UI *before* the user feels stressed.

---

## 1. Signals

- Micro-hesitation
- Looping
- Precision scanning
- Exit friction

---

## 2. Behaviour States

### absorb  
Engaged, reading well. Add optional detail.

### drift  
Light overwhelm. Simplify visuals.

### saturation  
High overwhelm. Chunk content.

### escape  
User preparing to leave. Offer save-and-exit.

---

## 3. Response Model

Predict → classify → adapt → reassure.
