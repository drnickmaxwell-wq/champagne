# Champagne Cognitive Load Engine

A system that gently reduces complexity when the user shows signs of overwhelm.

---

## 1. Activation Signals

Activated when user:
- pauses >3.5 seconds
- scrolls up repeatedly
- loops the same step
- interacts erratically
- expresses worry in chat
- reopens FAQ segments repeatedly

---

## 2. Adaptation Responses

### Visual
- reduce FX layers
- simplify background
- increase whitespace
- remove parallax

### Linguistic
- shorten sentences
- reduce jargon
- offer “Explain simply”

### Motion
- dampen shimmer
- disable bubbles
- slow caustics

### Navigation
- reveal breadcrumbs
- offer step segmentation
