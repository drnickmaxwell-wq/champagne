# Champagne Live Canvas Readiness V1

## Score

FAIL for safe manipulation.

## Why

Live Canvas needs explicit edit interfaces. Current repository has renderers and manifests, but no safe Live Canvas transaction model.

## Missing interfaces

- Section mutation API with allowlisted fields.
- Token-only style mutation validator.
- Preview-to-commit evidence protocol.
- Hero sacred-zone read-only adapter.
- Per-route manifest patch schema.
- Rollback and diff ledger.
